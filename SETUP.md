# 🌸 How to ship your dream GitHub profile

Your README will appear on your profile **only** if it lives in a repo whose name
**exactly matches your username**. For you that's `rabeyazaman/rabeyazaman`.


---

## 1 · Create the special repo

1. Go to https://github.com/new
2. **Repository name:** `rabeyazaman` (must match your username exactly)
3. ✅ Public
4. ✅ "Add a README file" → leave checked (GitHub will show a 🎉 banner saying
   "you found a secret!")
5. Click **Create repository**.

## 2 · Upload these two files

From this folder (`github-profile/`):

| File | Where it goes in the repo |
| --- | --- |
| `README.md` | repo root |
| `.github/workflows/snake.yml` | `.github/workflows/snake.yml` |

### Easiest way (browser, no terminal)

1. Open https://github.com/rabeyazaman/rabeyazaman
2. Click **Add file → Upload files**, drop `README.md`, commit.
3. Click **Add file → Create new file**.
   In the filename box type:&nbsp;&nbsp;`.github/workflows/snake.yml`
   (typing the slashes creates the folders automatically)
4. Paste the contents of `snake.yml`, commit.

## 3 · Trigger the snake animation once

The snake SVG is generated by a GitHub Action. It runs every 12 h, but for the
**first** time you need to kick it off manually:

1. Go to https://github.com/rabeyazaman/rabeyazaman/actions
2. Click **Generate Snake Animation** in the left sidebar.
3. Click **Run workflow → Run workflow** (green button).
4. Wait ~30 s. A new branch called `output` will appear holding the SVGs.

After this first run the snake on your profile will be live and self-updating.

## 4 · (Optional) personalize

Open `README.md` and tweak:

- The **About me** YAML block — change the focus / fun_fact / ask_me lines.
- The **Featured Projects** section — swap repo names if you create new ones.
- **Social links** — replace the LinkedIn/Facebook URLs with your real ones.
- Colors — search for `A78BFA` (purple) or `ec4899` (pink) and change the hexes.
